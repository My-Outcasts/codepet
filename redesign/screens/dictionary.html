Function

Functions

A function is a reusable, named block of code that performs a specific task. You define it once, then call it whenever you need it — optionally passing in values (parameters) and getting a value back (return value). Functions keep code organized, readable, and DRY (don't repeat yourself).

Example

def greet(name): return "Hi, " + name greet("Luna") # → "Hi, Luna"
